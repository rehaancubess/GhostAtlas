# GhostAtlas Product Overview

GhostAtlas is a horror-themed platform for recording, sharing, and exploring real-life paranormal encounters. Users can submit ghost stories with location data and images, which are then enhanced with AI-generated horror narratives, illustrations, and voice narration.

## Core Features

- **Encounter Submission**: Users submit paranormal stories with location, time, and optional images
- **Geospatial Discovery**: Browse encounters on an interactive map based on proximity
- **AI Enhancement**: Approved stories receive AI-generated horror narratives, illustrations, and voice narration using AWS Bedrock
- **Community Engagement**: Rate encounters (1-5 stars) and verify locations by visiting them
- **Admin Moderation**: Review and approve/reject submissions before publication
- **Ghostbuster Mode**: Special verification feature for users to check in at haunted locations

## Platform Components

1. **Mobile App** (Flutter): iOS/Android native experience with full feature set
2. **Web App** (React): Responsive web interface for browsing and submitting encounters
3. **Backend** (AWS Serverless): API Gateway, Lambda, DynamoDB, S3, Bedrock, Polly

## Design Aesthetic

Dark horror theme with:
- Pure black backgrounds (#000000, #0A0A0A)
- Eerie green accent color (#00FF41)
- Creepster font for headings
- Green glow effects, fog overlays, and atmospheric animations
