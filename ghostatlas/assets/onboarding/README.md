# Onboarding Illustration Assets

This directory contains the illustration assets for the GhostAtlas onboarding flow.

## Required Assets

All assets should be PNG format with transparency, optimized for mobile devices.
Recommended size: 512x512px (provide @2x and @3x versions for iOS if needed)

### 1. ghost_welcome.png

**Description:** Dark, mysterious welcome illustration featuring a ghostly figure or the GhostAtlas logo with paranormal elements
**Theme:** Welcoming but spooky, sets the horror atmosphere
**Suggested elements:** 
- Ghostly silhouette or translucent spirit
- Dark background with subtle fog
- Green glow accents
- GhostAtlas branding

## Design Guidelines

- **Color Palette:** Use dark backgrounds (#000000, #0A0A0A) with eerie green accents (#00FF41)
- **Style:** Horror-themed but not too scary, mysterious and atmospheric
- **Transparency:** Use PNG with alpha channel for flexible placement
- **Consistency:** Maintain consistent style across all 5 illustrations
- **Readability:** Ensure illustrations work well with overlaid text
- **Performance:** Optimize file sizes for mobile (aim for <200KB per image)

## Testing

After adding real assets:
1. Run `flutter pub get` to ensure assets are recognized
2. Test on both iOS and Android devices
3. Verify animations work smoothly with actual images
4. Check file sizes and loading performance
5. Ensure images look good on various screen sizes
