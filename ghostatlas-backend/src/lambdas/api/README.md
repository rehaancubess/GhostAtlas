# API Lambda Functions

This directory contains Lambda functions that handle API Gateway requests.

## Structure

- `submit-encounter/` - Handles encounter submission
- `get-encounters/` - Handles geospatial queries for encounters
- `get-encounter-by-id/` - Retrieves specific encounter details
- `rate-encounter/` - Handles rating submissions
- `verify-location/` - Handles location verification check-ins
- `admin-list-pending/` - Lists pending encounters for admin review
- `admin-approve/` - Approves encounters and triggers enhancement
- `admin-reject/` - Rejects encounters

Each Lambda function will be implemented in subsequent tasks.
