# Enhancement Pipeline Lambda Functions

This directory contains Lambda functions for the AI enhancement pipeline.

## Structure

- `orchestrator/` - Orchestrates the enhancement workflow
- `generate-narrative/` - Generates horror narrative using Bedrock
- `generate-illustration/` - Generates spooky illustration using Bedrock
- `generate-narration/` - Generates voice narration using Polly

Each Lambda function will be implemented in subsequent tasks.
